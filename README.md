# Trukos Github
ver diferencias
```
git diff
```

ver lo que tenemos local
```
git status
```

ver rama
```
git branch
```

Crear una rama
```
git checkout -b branchname
```

Eliminar una rama
```
git checkout -d branchname
```

Unir Rama
```
git checkout master
git merge branchname
```

ver el log
```
git log -3
```
```
git log -5 --pretty=oneline
```
